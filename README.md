## Cara Run
### Setup
Install seluruh requirement dari OS di dalam Linux anda
```bash
sudo apt update
sudo apt install -y nasm gcc qemu-system-x86 make genisoimage gdb
```
Agar dapat menjalankan OS, pastikan anda sudah berada di root repository ini
```bash
make start
```
